# Аттестация құжаттарын басқару жүйесі

## Қолданушы нұсқаулығы

### Админ панельге кіру

1. Браузерде ашыңыз: `admin.html`
2. Құпия сөзді енгізіңіз: `admin2024`
3. "Кіру" батырмасын басыңыз

### PDF файлдарын жүктеу

1. Сол жақ панельден санатты таңдаңыз
2. Егер санатта ішкі санаттар болса, оларды таңдаңыз
3. "PDF файлын таңдау" батырмасын басып, компьютерден файлды таңдаңыз
4. "Файлды жүктеу" батырмасын басыңыз

### Файлдарды көру және өшіру

1. Санатты таңдағаннан кейін, төменде барлық жүктелген файлдар көрсетіледі
2. "Қарау" батырмасы - файлды ашу үшін
3. "Өшіру" батырмасы - файлды жою үшін

## Техникалық ақпарат

### Файлдар құрылымы:

```
attestation/
├── pdfs/              # PDF файлдар сақталатын папка
├── data.json          # Құжаттар мәліметтер базасы
├── admin.js           # Админ панель логикасы
├── viewer.js          # Қолданушы интерфейсі
└── README.md          # Бұл файл
```

### Құпия сөзді өзгерту:

`attestation/admin.js` файлында:
```javascript
const ADMIN_PASSWORD = 'admin2024'; // Осы жолды өзгертіңіз
```

### Деректерді сақтау:

Файлдар туралы ақпарат екі жерде сақталады:
1. `localStorage` - браузерде
2. `data.json` - файлда (қолмен backup жасауға болады)

### Қауіпсіздік ескертулері:

⚠️ Бұл жүйе демо нұсқа. Өндірістік қолдану үшін:
- Бэкенд сервер қажет (Node.js, PHP және т.б.)
- Файлдарды серверде сақтау
- Қауіпсіз аутентификация жүйесі
- HTTPS протоколы

## Жиі қойылатын сұрақтар

**С: Файл жүктелген жоқ, не істеу керек?**
Ж: Файл PDF форматында екенін тексеріңіз. Тек .pdf файлдарын жүктеуге болады.

**С: Файлдар жоғалды, қалай қалпына келтіруге болады?**
Ж: Браузер кэшін тазаламаңыз. Деректер localStorage-те сақталады.

**С: Құпия сөзді ұмыттым?**
Ж: `attestation/admin.js` файлында жаңа құпия сөз орнатыңыз.
